Projects

I created playpicturepoker.com a simplified poker game where the player goes against the dealer for high stakes betting! The site was created using HTML, CSS, and Javascript. The game involves having an array for the player and dealers hands, the ability to redraw cards for the player and dealer, a matching and priority determiner for figuring out the winner, and a bet/money system. It's a simple but addictive game that I am still adding on to.



The summer after I learned Python and PyGame I made a 2D block based shooting game. Some complex coding problems I dealt with in this project include: text file based level creation, 2D array visualized as the gameboard, enemy classes with a variety of abilities, and music/sound integration.

For my final project in my Computer Graphics course, I made a model of a car driving on a circle road with day/night cycle lighting. This was made with mainly javascript and some html.
